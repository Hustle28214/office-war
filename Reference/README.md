# 参考答案（/Reference）

本目录在 **Assets 外**，Unity **不会编译**，避免与你在 `Assets/_Project/Scripts/` 写的类重名冲突。

## 使用规则

1. **先按 [`教案.md`](../教案.md) 自己写**  
2. 卡关再看同路径文件，例如 `Reference/Battle/BattleManager.cs`  
3. 不要整文件复制到 Assets  

## 目录对应

| 你写的 | 参考答案 |
|---|---|
| `Assets/_Project/Scripts/Core/GameEnums.cs` | `Reference/Core/GameEnums.cs` |
| `Assets/_Project/Scripts/Battle/BattleManager.cs` | `Reference/Battle/BattleManager.cs` |

Perks / OA 无完整参考，按教案 + plan 实现。
