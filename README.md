# 明天能上线吗

> 🎓 **Unity 学习项目** — 跟 [`教案.md`](教案.md) 从课 01 做到课 20。  
> 协作策略卡牌：钉钉感工作台 UI · 德式 KPI · 职位卡包 · 内网开房。

## 快速开始

1. Unity 2022.3 LTS+ 打开本项目  
2. 阅读 **[`教案.md`](教案.md)** 课 01（先读 [`plan.md`](plan.md) §一~§三）  
3. 从 **课 02** 打开 `Scripts/Core/GameEnums.cs` 开始写  
4. **课 17** 完成后 Play `Assets/_Project/Scenes/Battle.unity`  
5. 菜单：**明天能上线吗 → Setup Demo Scene**

当前 Play 只会提示「学习模式」——这是预期行为。
