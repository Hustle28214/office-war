# 疯狂周一：办公室战争

> 🎓 **Unity 学习项目** — 跟 [`教案.md`](教案.md) 从课 01 做到课 20，亲手实现完整 Demo。

## 快速开始

1. Unity 2022.3 LTS+ 打开本项目  
2. 阅读 **[`教案.md`](教案.md)** 课 01  
3. 从 **课 02** 打开 `Scripts/Core/GameEnums.cs` 开始写代码  
4. **课 17** 完成后 Play `Assets/_Project/Scenes/Battle.unity`

当前 Play 只会提示「学习模式」——这是预期行为。
